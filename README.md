## ğŸ› ï¸ Installation & Setup

If you've downloaded this project and want to run it on your computer, follow these steps:

### Prerequisites
- **Node.js**: You need to have Node.js installed. [Download it here](https://nodejs.org/).

### Steps
1.  **Open a Terminal**: Navigate to the project folder.
    ```bash
    cd forest-fairy-gemstone-game
    ```

2.  **Install Dependencies**: Run the following command to install the necessary libraries.
    ```bash
    npm install
    ```

3.  **Start the Game**: Launch the local development server.
    ```bash
    npm run dev
    ```
    Click the link shown in the terminal (usually `http://localhost:5173`) to play!

4.  **Build for Production** (Optional): To create a optimized version for deployment.
    ```bash
    npm run build
    ```
